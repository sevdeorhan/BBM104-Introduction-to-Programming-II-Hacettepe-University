# ☕ Hacettepe University - BBM 104: Introduction to Programming Laboratory II

This repository contains a collection of Java projects developed during the **BBM 104** course at **Hacettepe University** (Spring 2025). The projects focus on advanced **Object-Oriented Programming (OOP)** principles, GUI development, and complex system design.

## 🚀 Projects Overview

| Project | Description | Core Concepts |
| :--- | :--- | :--- |
| **[Library Management](./BBM104-PA1)** | A system managing users and items with specific borrowing rules. | Inheritance, Encapsulation |
| **[Zoo Manager](./BBM104-PA2)** | A simulation of zoo operations, feeding logic, and personnel roles. | Abstraction, Polymorphism |
| **[Student Management](./BBM104-PA3)** | A platform managing university entities, grades, and GPA reports. | Interfaces, Collections (Map/List) |
| **[2D Tank Battle Game](./BBM104-PA4)** | A real-time interactive game with physics and animations. | JavaFX, Game Loop, Collision Logic |
